# Aira AI (uzumaki-ai)

Singkat: Aira AI adalah aplikasi chat AI statis berbasis browser. Aplikasi ini adalah halaman statis (index.html) yang berisi HTML, CSS, dan JavaScript untuk UI chat, pemilihan provider/model, penyimpanan API key lokal, dan pengelolaan riwayat percakapan.

Instalasi & Menjalankan

1. Clone repo

   git clone https://github.com/acukgelogor115-ux/uzumaki-ai.git
   cd uzumaki-ai

2. Jalankan server statis sederhana atau buka index.html langsung di browser:

   # Python 3
   python3 -m http.server 8000
   # buka http://localhost:8000

Catatan Keamanan

- API key disimpan secara lokal di browser (localStorage). Jangan gunakan API key produksi di deploy publik tanpa proxy server yang aman.
